This is a chair package for spawning chairs.


The package currently (v1.1) includes 60 pre-made Chair Models.

::: Update v1.1 :::


Added 20 more chairs.


Added HP/SP regen to the chairs in the DataSet.


Added Emote support with the new event FaceChangeEvent.

How to use:


After importing the package, add ChairComponent to the DefaultPlayer and set the ChairID.


The ChairID is the ID of the specific chair found in the DataSet "Chair" within the DataSets folder.


If you want to change the players chair while the game is running, use the SetChairID() method in the ChairComponent.
